Hello! My name is Nikos Baxevanis, and I'm a full-stack programmer working on small as well as big projects using test-driven development. I live in Glyfada and work out of Pisia, Greece.

Some projects

Its primary goal is to allow developers to focus on what is being tested rather than how to setup the test scenario, by making it easier to create object graphs containing test data.
